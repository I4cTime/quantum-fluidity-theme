A crisp, high-contrast dark theme with luminous accents, designed for a development environment that feels both quantum and agentic.

## Palette

- **Obsidian Void:** `#050505` (Backgrounds)
- **Electric Cyan:** `#00D1FF` (Borders, Highlights, Strings, Functions)
- **Hyper-Violet:** `#B266FF` (Badges, Notifications, Keywords, Control Flow)
- **Midnight Mist:** `#1A1A2E` (Hover States, Selections)
- **Ghost White:** `#F8F9FA` (Foreground Text, Variables)

## Usage

Enjoy the subtle glow and sharp readability of this "Dark Mode" aesthetic, bringing fluid motion and agentic vibes to your editor.
